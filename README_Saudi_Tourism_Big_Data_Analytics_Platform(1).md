# 📌 Overview

This project presents an end-to-end Big Data analytics platform for
analyzing tourism reviews collected from Google Maps across Saudi
Arabia.

The platform demonstrates how distributed data processing, Natural
Language Processing (NLP), and Business Intelligence can be combined to
transform millions of tourism reviews into actionable insights for
tourism decision-makers.

The project was developed as part of the **Big Data Mining** course.

# 🏗 Project Architecture

The platform follows a distributed Big Data pipeline consisting of data
ingestion, Apache Spark processing, sentiment analysis, and interactive
dashboard visualization.
