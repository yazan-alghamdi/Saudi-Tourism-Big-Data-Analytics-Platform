# Saudi Tourism Big Data Analytics Platform

## 🌍 Overview

This project presents a scalable **Big Data Analytics Platform** developed to process, analyze, and visualize massive tourism review datasets collected from destinations across Saudi Arabia.

Using **Apache Spark**, **HDFS**, **PySpark**, and **Power BI**, the platform processes more than **1.2 million reviews**, enabling distributed analytics, Arabic NLP, and interactive dashboards that support tourism decision-making aligned with **Saudi Vision 2030**.
